Throughout this module, I learned basic linux commands, how to navigate files and directories, how to navigate Windows, and basic Windows commands, as well as use various Linux commands
in the context of real-world applications with the use of OverTheWire's levels 1-15. Through this course I learned how to open files, find file locations, navigate around odd file names,
changing directories, filtering for files by certain criteria, including text, file size, group and user, as well as decoding and decompressing files to reveal hidden texts within. Lastly,
I learned how to connect and gain access to remote machines.
